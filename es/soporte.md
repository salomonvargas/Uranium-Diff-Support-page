# Soporte

¿Necesitas ayuda, encontraste un fallo o tienes una idea? Estos son los canales:

### 📮 Contacto directo
Correo: **uraniumdiff@outlook.com**
Responderemos lo antes posible. Para agilizar, incluye tu versión de macOS y la versión de la app (menú **Uranium Diff ▸ Acerca de**).

### 🐞 Reportar un error
Abre un ticket en nuestro **rastreador público de GitHub Issues** (el mismo repositorio de esta página; el código de la app se mantiene aparte y privado):
👉 `https://github.com/salomonvargas/Uranium-Diff-Support-page/issues/new/choose`

Para un buen reporte, incluye:
1. Qué esperabas que pasara y qué pasó.
2. Pasos para reproducirlo.
3. Tipo de archivo/formato y origen (local, URL, SSH).
4. Versión de la app y de macOS; modelo de Mac.
5. Capturas de pantalla si aplica (evita incluir información confidencial).

### 💡 Sugerir una función
¿Te falta algo? Ábrelo también como **GitHub Issue** con la etiqueta *feature request*, o escríbenos por correo.

### 📚 Antes de escribir
Quizá tu duda ya esté resuelta en:
- [Inicio rápido](inicio-rapido.md)
- [Preguntas frecuentes](preguntas-frecuentes.md)
- [Solución de problemas](solucion-de-problemas.md)

> Nota de privacidad: los GitHub Issues son **públicos**. No incluyas contraseñas, rutas privadas ni contenido confidencial de tus archivos.
