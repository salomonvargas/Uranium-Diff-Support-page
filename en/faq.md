# Frequently asked questions

**Is Uranium Diff free?**
Yes — the **Free plan is permanent** and includes text/code comparison and structural diff. Advanced features (folders, SSH, office formats, merge, Git, AI, export, CLI) make up **Uranium Diff Pro**, a subscription. Every install includes a **14-day trial** of all Pro features, no card required.

**Are my files uploaded to the cloud?**
No. All processing happens **on your Mac**. Your file contents are never sent to any server. The only outbound network happens when **you** download from a URL or connect to an **SSH** server you configure.

**Which formats can I compare?**
Text and code, JSON/XML/plist/YAML (structural), PDF, Word (.docx), Excel (.xlsx), and images (PNG/JPEG/HEIC…).

**Can I edit or save files from the app?**
Uranium Diff is a diff **viewer**: it doesn't edit file contents. In **folder** mode you can **copy** files between sides.

**My SSH key doesn't work — why?**
**Key-based** authentication supports **Ed25519** and **ECDSA** keys (with or without a passphrase). **RSA** keys are **not** supported for key auth. For RSA-only servers, use **password** authentication.

**I don't see the Apple Intelligence features.**
Uranium Diff requires **macOS 26 (Tahoe) or later** on an **Apple Silicon** Mac. Also check that **Apple Intelligence** is **enabled** in System Settings. If unavailable, the app says so and everything else keeps working.

**Can I compare files that aren't UTF-8?**
The current version assumes **UTF-8**. Files in other encodings may display incorrect characters. Encoding selection is on the roadmap.

**Does comparing modify my files?**
No. Comparing (including structural comparison) **never** alters your original files.

**Do I need an internet connection?**
Only to compare from a **URL** or over **SSH**. Local comparison and the on-device AI features work **offline**.

**Is there a file-size limit?**
Downloads from a URL are capped at **50 MB**. Very large local files may take longer; cancel any time with **⌘.**

**Is there an iPhone or iPad version?**
No. Uranium Diff is **macOS-only**.

**How do I manage or cancel my Pro subscription?**
From your **App Store account settings** (Apple ID ▸ Subscriptions). It auto-renews unless canceled at least 24 hours before the period ends.

**How do I restore a purchase on another Mac?**
Sign in with the same Apple ID and use **"Restore Purchases"** on the Uranium Diff Pro screen.
