# Support

Need help, found a bug, or have an idea? Here's how to reach us:

### 📮 Direct contact
Email: **uraniumdiff@outlook.com**
We'll reply as soon as we can. To speed things up, include your macOS version and the app version (**Uranium Diff ▸ About**).

### 🐞 Report a bug
Open a ticket in our **public GitHub issue tracker** (the same repository as this page; the app's source code is kept separate and private):
👉 `https://github.com/salomonvargas/Uranium-Diff-Support-page/issues/new/choose`

For a good report, include:
1. What you expected vs. what happened.
2. Steps to reproduce.
3. File type/format and source (local, URL, SSH).
4. App and macOS versions; Mac model.
5. Screenshots if relevant (avoid confidential information).

### 💡 Request a feature
Missing something? Open a **GitHub Issue** with the *feature request* label, or email us.

### 📚 Before you write
Your question may already be answered in:
- [Quick start](quick-start.md)
- [FAQ](faq.md)
- [Troubleshooting](troubleshooting.md)

> Privacy note: GitHub Issues are **public**. Don't include passwords, private paths, or confidential file contents.
